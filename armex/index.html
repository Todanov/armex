Статусы заказов и штрихкодирование

В программе 1С:УНФ 8. Полиграфия 2 реализована возможность использования дополнительного торгового оборудования — сканер штрих-кода.
С помощью данной обработки можно автоматически производить 3 операции:
Смена статусов заказа по штрих-коду статуса Поиск документов по штрих-коду на печатной форме расчет заказа Смена статусов заказа с идентификацией сотрудника по штрих-коду сотрудника
